# Workflow Next.js

A container for various Next.js based workflow projects and components.

## Featured Projects

### [Work-Flow](work-flow/)
A sophisticated Next.js dashboard and workflow application.
- **Frontend**: Next.js, TypeScript, Tailwind CSS.
- **Components**: Uses Radix UI / Shadcn based components (Accordion, Alert, Button, etc.).
- **Architecture**: Dashboard layout with global styles and custom hooks.

## Key Features
- **Dashboard Layout**: Pre-built dashboard structure.
- **UI Components**: A rich set of reusable UI components.
- **TypeScript**: Ensuring type safety across the project.

## Getting Started
Navigate to the `work-flow` directory to install dependencies and run the development server:
```bash
cd work-flow
npm install
npm run dev
```
